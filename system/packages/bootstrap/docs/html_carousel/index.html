Html Carousel plugin

This plugin requires bootstrap-carousel.js

carousel( array(attrs) )

  1. slide => (bool) animate transitions (default: false)
  2. interval => (int) Interval (seconds) beetwenn slides (default 2000)
  3. pause => (int) Pauses the cycling of the carousel (default: hover)
  4. active => (int) Manually define the index of the active slide (default 0)

If you pass an id attibute, content-panes will use as namepace

add( text, title, content )

add a slice.
If title and content are provided, the method set() will be called

caption( title, content)

Set the caption content of the prev added slice

active( (int) index )

Set the active index of the set

render( )

Returns the html

Fully configured example example

Html::carousel(array('slide' => true, 'active' => 1, 'interval' => 3000))
		->add(Html::img('bootstrap-mdo-sfmoma-01.jpg'))->caption('Title 1', 'First caption content')
		->add(Html::img('bootstrap-mdo-sfmoma-02.jpg'))->caption('Title 2', 'Second caption content')
		// dont use caption()
		>add(Html::img('bootstrap-mdo-sfmoma-03.jpg'), 'Title 3', 'Third caption content')
		->render();
